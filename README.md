# Ilac Analiz Asistani

Fotograf veya kamera ile ilac kutusunu tara, AI ile detayli analiz al.

## Kurulum ve Calistirma

1. Projeyi Indirin: `git clone https://github.com/nursenaunalan/ilac_projesi.git`
2. Bagimliliklari Yukleyin: `pip install -r requirements.txt`
3. API Anahtarlerini .env veya Streamlit Secrets uzerinden ayarlayin (GROQ_API_KEY, GEMINI_API_KEY).
4. Calistirin: `streamlit run app.py`

## Canli Demo
Uygulama Streamlit Cloud uzerinde yayindadir.

## Yasal Uyari
Bu uygulama egitim amaclidir. Tibbi teshis veya tedavi tavsiyesi degildir. Ilac kullanmadan once her zaman doktorunuza veya eczaciniza danisiniz.
